<Typography variant="h3" backgroundColor="gray">
  <span>사용법</span>
</Typography>

- VSCode Bookmark 단축키
  - 북마크 지정 / 해제  
     `[Ctrl]` + `[Alt]` + `K`
  - 이전 북마크로 이동  
     `[Ctrl]` + `[Alt]` + `J`
  - 다음 북마크로 이동  
     `[Ctrl]` + `[Alt]` + `L`
- 이전, 다음 북마크로 이동 시 모든 파일의 북마크를 순차적 이동하는데  
   열려있는 파일만 북마크 이동하고 싶다면  
   설정에서 `bookmarks: Navigate Through All Files` 검색하여 체크해제!
- 추가 관련 기능 검색은 `F1` 누르고 _Bookmarks_ 입력

<Typography variant="h3" backgroundColor="gray">
  <span>Sitemap 등록 (웹 마스터 도구에 등록)</span>
</Typography>
<br />

**[Google Search Console](https://search.google.com/search-console/about)**

1. 로그인 → 시작하기 → **URL 접두어** 옵션에서 **[자신의 URL]** 입력
2. 소유권 확인 작업
   - **HTML 태그 등록 방법** → [웹 마스터 도구에 블로그 등록하기](https://www.notion.so/c7b05ba85dc043babb8428fccb4070e2) 글 참고
   - [Google Domains](https://domains.google/)에서 도메인을 샀다면..? → 자동으로 소유권 확인됨.
3. 좌측 메뉴에서 **Sitemaps** 탭 클릭 후, Sitemap 등록
   - 보통 `sitemap.xml`의 위치는 <Typography color="gray600" isBold>https://[DomainName]/sitemap.xml</Typography>에 있음

[**Naver Search Advisor**](https://searchadvisor.naver.com/)

1. 로그인 → 웹 마스터 도구 클릭 → 사이트 등록란에서 **[자신의 URL]** 입력
2. 소유권 확인 작업
   - **HTML 태그 등록 방법 (**Google의 HTML 태그 등록 방법과 거의 동일**)**
3. 등록된 URL 클릭 → 좌측 메뉴의 **요청** 탭 → **사이트맵 제출 탭** → Sitemap 등록 (⬆️ <Typography color="gray500">Google Search Console</Typography> 방법과 같음)
